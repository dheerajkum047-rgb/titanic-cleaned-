# 📊 Data Cleaning & Missing Value Handling Project

## 📌 Overview

This project focuses on data cleaning and handling missing values using Python. It demonstrates how raw datasets can be transformed into clean and usable data for analysis and machine learning tasks.

---

## 🎯 Objective

* Identify missing values in the dataset
* Handle missing data using different techniques
* Perform data cleaning operations
* Ensure dataset quality for further analysis

---

## 🗂️ Dataset Description

The dataset contains student-related information including:

* Personal details (Name, Age, Gender, City)
* Academic details (Class, Marks)
* Attendance records

Some values are intentionally missing to simulate real-world data problems.

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Jupyter Notebook / VS Code

---

## 🔍 Data Cleaning Steps

* Checked missing values using `isnull().sum()`
* Filled missing values using:

  * Mean (for numerical data)
  * Mode (for categorical data)
* Dropped unnecessary columns (if required)
* Verified cleaned dataset

---

## 📊 Key Operations Performed

* Missing value analysis
* Data imputation (mean/median/mode)
* Data filtering and cleaning
* Dataset validation after cleaning
* Optional visualization using heatmaps

---

## 📁 Project Files

* `student_dirty.csv` → Raw dataset with missing values
* `data_cleaning.ipynb` → Python notebook for cleaning process
* `cleaned_dataset.csv` → Final cleaned dataset

---

## ▶️ How to Run

1. Install required libraries:

3. Run all cells step-by-step
